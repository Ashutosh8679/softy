# Tinkercad Simulation Guide

This folder contains a browser-based simulation created using Tinkercad.

## 📂 How to Access the Simulation

Follow these steps:

1. Open the file named `open_simulation.html` in this folder.
2. Double-click the file (or right-click → Open).
3. Your default web browser will launch automatically.
4. Click on the link shown on the page to open the simulation.

## ▶️ Running the Simulation

1. Once redirected to Tinkercad, sign in if required.
2. Click on **"Start Simulation"**.
3. Interact with the circuit/components as needed.

## ⚠️ Notes

- Internet connection is required.
- If the project opens in **view-only mode**, use the **"Duplicate"** or **"Remix"** option to edit it.
