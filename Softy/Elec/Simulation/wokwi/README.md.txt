# Wokwi Simulation Guide

This folder contains a browser-based simulation created using Wokwi.

## 📂 How to Access the Simulation

Follow these steps:

1. Open the file named `open_simulation.html` in this folder.
2. Double-click the file (or right-click → Open).
3. Your default web browser will open.
4. Click on the link displayed on the page.

## ▶️ Running the Simulation

1. The Wokwi project will load automatically.
2. Click the **"Play" (▶️)** button to start the simulation.
3. You may edit the code or components if required.

## ⚠️ Notes

- No login is required for basic usage.
- You can **fork/save** the project if you want your own editable version.
